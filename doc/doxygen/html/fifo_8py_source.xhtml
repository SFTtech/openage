14 class FIFO(FileLikeObject):
15  """
16  File-like wrapper around ByteQueue.
17 
18  Data written via write() can later be retrieved via read().
19 
20  EOF handling is a bit tricky:
21 
22  EOF in the data source (the writer) can not be auto-detected, and must
23  be manually indicated by calling seteof().
24 
25  Only then will read() show the desired behavior for EOF.
26 
27  If EOF has not yet been set, read() will raise ValueError if more data
28  than currently available has been requested.
29  """
