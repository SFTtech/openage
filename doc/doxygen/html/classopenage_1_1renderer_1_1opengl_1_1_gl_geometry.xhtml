void openage::renderer::opengl::GlGeometry::update_verts_offset ( std::vector< uint8_t > const &  verts,
size_t  offset 
)
overridevirtual

In a meshed geometry, updates the vertex data starting from the offset-th vertex. The type of the vertex data has to be the same as it was on initializing the geometry. The size plus the offset cannot exceed the previous size of the vertex data. If the mesh is indexed, indices will stay the same.

Exceptions
ifthere is a size mismatch between the new and old vertex data
