21 class FSLikeObject(ABC):
22  """
23  To be implemented by any filesystem-like objects that wish to provide
24  their contents via Path-like and file-like objects.
25 
26  Normally, you use the Path provided by self.root for example!
27 
28  The abstract member methods take a list or tuple of path component
29  bytes objects, e.g.: [b'etc', b'passwd'].
30 
31  If a request can not be fulfilled for whatever reason (file doesn't exist,
32  the object is read-only, the given method is not implemented, ...),
33  they may and shall raise an appropriate instance of IOError.
34  """
