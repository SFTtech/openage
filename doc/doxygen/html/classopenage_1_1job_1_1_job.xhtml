Detailed Description

template<class T>
class openage::job::Job< T >

A job is a wrapper around a shared job state object and is returned by the job manager. It can be used to retrieve the current state of the job and its result. A job is a lightweight object which only contains a pointer to its internal shared state. Thus it can be copied around without worrying about performance. Further it is not necessary to create or pass pointers to job objects.
