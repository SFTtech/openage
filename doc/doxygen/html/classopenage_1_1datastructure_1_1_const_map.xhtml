Detailed Description

template<typename K, typename V, size_t pos>
class openage::datastructure::ConstMap< K, V, pos >

Compiletime generic lookup map.

This is a recursive template that decreases pos which each iteration step. Each pos stores one map entry. The lookup recursively descends those entries down to the zero-element.

If you experience compiler errors, make sure you request existing keys. We intentionally trigger compiler failures when a key doesn't exist.

Messages include: "error: ‘*0u’ is not a constant expression" -> nonexistant key
