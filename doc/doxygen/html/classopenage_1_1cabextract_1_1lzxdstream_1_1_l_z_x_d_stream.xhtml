Detailed Description

Read-only stream object that wraps LZXDecompressor.

Constructor arguments:

Parameters
compressed_fileThe compressed file-like object; must implement only read(). If seek(0) works on it, reset() works on this object.
window_bitsProvided as metadata in MSCAB files; see LZXDecompressor.

Defaults to 21.

Parameters
reset_intervalZero for MSCAB files; see LZXDecompressor.

Theoretically, if reset_interval > 0, efficient seek() could be implemented. However, it isn't.

Defaults to 0.
