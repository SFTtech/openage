◆ get_native_path()

std::string openage::util::Path::get_native_path ( ) const
private

Return the native path (something like /your/folder/with/file) for this path. returns emptystring ("") if there is no native path.

This does not minimize/flatten (= "resolve") the path!

You probably want to use resolve_native_path instead.

Resolve the native path by flattening all underlying filesystem objects (like unions). Returns the /native/path/on/disk. Throws up if there is no native path.

This basically is the same as resolve_*().get_native_path().
