overridevirtual

Returns a pointer to the sample buffer at the given position and the number of int16_t values that are actually available. If the end of the resource is reached, 0 will be returned. If the resource is not ready yet, a nullptr with a length, different to zero, will be returned.

Parameters
positionthe current position in the resource
data_lengththe number of int16_t values that should be returned
data_lengththe number of bytes of audio data that is requested

◆ decay_queue

datastructure::ConcurrentQueue<std::shared_ptr<chunk_info_t> > openage::audio::DynamicResource::decay_queue
private

Queue of audio chunk information that manages the decay of buffer data.

This implements "forget least-recently-used chunks" when a new one shall be loaded. This must never run empty! Otherwise the audio system will lock up.
