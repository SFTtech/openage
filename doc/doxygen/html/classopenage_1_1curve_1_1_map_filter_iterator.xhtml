Detailed Description

template<typename key_t, typename val_t, typename container_t>
class openage::curve::MapFilterIterator< key_t, val_t, container_t >

A filtering operator to iterate over all elements of a map whose elements exist for a certain livespan. The range where to iterate is given at construction.

It depends on key_t and val_t as map-parameters, container_t is the container to operate on and the function valid_f, that checks if an element is alive.
